# 🎙️ Voice-to-Text Recorder (Whisper)

Программа в данный момент не стабильна, предпринимаются попытки починки. При следующем обновлении я укажу чёткий список библиотек которые нужно установить, а так же поменяю файлы на актуальные.

Голосовой диктофон с распознаванием речи в реальном времени на базе **Faster-Whisper**.
Работает полностью локально, без подключения к интернету.

## Возможности

- Запись с микрофона и преобразование речи в текст.
- Глобальная горячая клавиша для старта/остановки записи.
- Системный трей (иконка в области уведомлений).
- Настройка устройства (CUDA / CPU), размера луча (beam size), усиления, нормализации, шумоподавления, изменения темпа речи.
- Автоматическая остановка записи при длительной тишине.
- Озвучивание статусов через синтезатор речи (SAPI5).
- Копирование результата в буфер обмена.

## Системные требования

### Обязательные компоненты

- **Операционная система**: Windows 10 / 11 (из-за использования Windows API для трея и горячих клавиш).
- **Python** 3.8 – 3.11 (рекомендуется 3.11).
  При установке Python обязательно отметьте опцию **«Add Python to PATH»**.
- **Microsoft Visual C++ Redistributable** (последняя версия) – требуется для работы некоторых библиотек (например, `faster-whisper`, `numpy`, `sounddevice`).
  Обычно уже установлена, но если нет – скачайте с официального сайта Microsoft.

### Для работы на GPU (ускорение)

- **Видеокарта NVIDIA** с поддержкой CUDA (Compute Capability 5.2+).
- **CUDA Toolkit** версии 11.8 или 12.x (совместимая с вашим драйвером).
- **cuDNN** для соответствующей версии CUDA (обычно устанавливается вместе с драйверами или через пакетный менеджер).
- Драйвер NVIDIA не ниже 525.xx (для CUDA 11.8) или 535.xx (для CUDA 12.x).

**Важно:** если у вас нет NVIDIA GPU или вы не хотите устанавливать CUDA, вы можете работать на **CPU** – выберите в настройках `device = cpu`. Однако учтите, что на CPU распознавание может быть значительно медленнее.

### Опциональные компоненты

- **FFmpeg** – не является обязательным для работы программы, так как используется raw-аудио с микрофона. В текущей версии не требуется.
- **pyrubberband** – для изменения темпа речи (time-stretch). Если модуль не установлен, функция будет недоступна.
- **noisereduce** – для шумоподавления. Если модуль отсутствует, эта опция не будет работать (но программа запустится).

## Установка

1. **Скачайте проект** (архив или через `git clone`).

2. **Откройте командную строку** в папке проекта и выполните:

   ```bash
   pip install -r requirements.txt
   ```

3. Если вы планируете использовать GPU, убедитесь, что в системе установлены CUDA и cuDNN, а также что библиотека `faster-whisper` правильно их подхватит (обычно это происходит автоматически).

4. Запустите программу:
   - Через командную строку: `python main.py`
   - Или двойным щелчком по `start.bat` (если Python в PATH).

### Первый запуск

При первом запуске будет автоматически загружена модель `large-v3-turbo` (размер ~3.5 ГБ). Это может занять некоторое время и потребовать места на диске (модель сохраняется в кэш Hugging Face, обычно в папке пользователя `~/.cache/huggingface`).

После загрузки модели в трее появится зелёная иконка, и программа будет готова к работе.

## Настройка

Нажмите кнопку ⚙️ в главном окне для открытия панели настроек.

Вы можете изменить:

- Устройство (`cuda` / `cpu`).
- Размер луча (beam size) – влияет на точность и скорость.
- Горячую клавишу для старта/остановки (формат: `f9`, `ctrl+shift+r`, `left alt+caps lock`).
- Параметры обработки аудио (усиление, нормализация, шумоподавление, темп).
- Автоматическую остановку по тишине.

Настройки сохраняются в файл `settings.json` в папке проекта.

## Автозапуск

При включении опции «Автозапуск с системой» программа создаёт `.bat`-файл в папке автозагрузки Windows.
Для этого она автоматически находит `pythonw.exe` (через `shutil.which` или замену в `sys.executable`). Если Python не добавлен в PATH, автозапуск может не сработать – тогда укажите полный путь к `pythonw.exe` вручную в файле `start.bat` или добавьте Python в переменные среды.

## Устранение неполадок

- **Ошибка `pythonw.exe not found` при включении автозапуска** – убедитесь, что Python добавлен в PATH, либо установите соответствующую галочку при установке Python.
- **Не работает горячая клавиша** – проверьте, не занята ли комбинация другой программой. Попробуйте сменить хоткей в настройках.
- **Медленная работа на CPU** – используйте меньшую модель (например, `medium` или `small`) и уменьшите beam size.
- **Нет звука с микрофона** – проверьте, что микрофон выбран в системе как устройство ввода по умолчанию, и разрешите доступ к микрофону (если запрашивается).
- **Сбой при импорте `pynvml`** – если у вас нет GPU, установка `pynvml` необязательна (он используется только для мониторинга). Ошибка не критична.
